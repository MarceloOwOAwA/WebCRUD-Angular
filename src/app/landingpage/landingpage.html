Bienvenido a mi pagina de demostracion para creacion y login de usuarios

Esta es una aplicacion de ejemplo para demostrar la creacion y login de usuarios. aqui se conectara con otro docker que contiene API y Mongodb7 para realizar las operaciones CRUD, asegurando ademas seguridad informatica y autenticacion de usuarios con JWT.

Para comenzar es necesario que crees una cuenta utilizando la pagina o la API en el otro contenedor, para facilidad de uso dejare el link para que puedas descargar el contenedor del API aqui: github.com
